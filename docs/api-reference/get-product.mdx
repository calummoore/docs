---
title: "Get Product"
api: "GET https://client.corrily.com/v1/products/{product_api_id}"
description: "Retrieve the details of a single Corrily product."
---

<RequestExample>

```bash Response

curl --location --request GET 'https://default.corrily.com/mainapi/v1/products/monthly'
```

</RequestExample>

## Path Params
