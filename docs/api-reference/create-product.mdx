## Body Params



<Param body="name" type="string" required>

The name of the product. Must be unique.

</Param>

<Param body="base_px" type="float">

The base price of the product in USD.

</Param>

<Param body="interval" type="string" required>

A string value denoting how often to bill for a subscription. Possible values: `day`, `week`, `month`, `year`, `one_time`.

</Param>

<Param body="interval_count" type="int32" required>

An integer denoting how many intervals to bill for the. A monthly product would have `month` as interval and `1` as interval counts, etc.
One time products should have an interval count of `1`.

</Param>
